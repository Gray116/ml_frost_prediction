# Notebooks Directory

## 개요
이 디렉토리는 데이터 분석, 모델 프로토타이핑, 실험 등을 위한 Jupyter 노트북들을 포함.

EDA 과정에서 발견한 인사이트와 결정한 전처리 방법들은 나중에 src/ml_project/data/preprocessing.py에 구현

## 노트북 구조
### 1. EDA.ipynb
- 탐색적 데이터 분석(Exploratory Data Analysis)
- 데이터 품질 평가
- 전처리 방법 실험
```python
# 주요 분석 내용
1. 기본 데이터 탐색
   - 데이터 크기, 형태
   - 데이터 타입
   - 기술 통계
   - 결측치 현황

2. 데이터 시각화
   - 분포 확인
   - 상관관계 분석
   - 이상치 탐지

3. 전처리 실험
   - 결측치 처리
   - 이상치 처리
   - 특성 스케일링
```

### 2. model_prototype.ipynb
 - 모델 프로토타입 개발
 - 간단한 모델 실험
 ```python
 # 주요 내용
1. 데이터 로드 및 전처리
2. 모델 아키텍처 설계
3. 초기 학습 실험
4. 성능 평가 및 시각화
 ```


 ## 사용 가이드라인
1. 노트북 실행 순서
 - EDA.ipynb 먼저 실행
 - 전처리 방법 결정 후 model_prototype.ipynb 실행

2. 데이터 경로
 - 원본 데이터: ../data/raw/
 - 전처리된 데이터: ../data/processed/

3. 모범 사례
 - 각 셀에 마크다운으로 설명 추가
 - 중요한 발견 사항 문서화
 - 실험 결과 기록
 - 시각화 결과 저장

4. 주의사항
 - 노트북의 실험 결과는 src/ 디렉토리에 구현하기 전 프로토타입용
 - 대용량 출력은 적절히 축소하여 표시
 - 실행 순서 꼭 지키기
